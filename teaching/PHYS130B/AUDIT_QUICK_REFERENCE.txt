KEY EQUATIONS PRESENT IN BOTH SOURCES
================================================================================

✓ Density matrix: ρ = Σ_i p_i |ψ_i⟩⟨ψ_i|
✓ Purity: Tr(ρ²) = 1 (pure), < 1 (mixed)
✓ Partial trace: ρ_A = Tr_B(ρ_AB)
✓ Von Neumann entropy: S(ρ) = -Σ_i λ_i ln λ_i
✓ Schmidt decomposition: |Ψ⟩ = Σ_k λ_k |u_k⟩_A |v_k⟩_B
✓ Bell inequality (CHSH): |⟨A₁B₁⟩ + ⟨A₁B₂⟩ + ⟨A₂B₁⟩ - ⟨A₂B₂⟩| ≤ 2
✓ Bloch vector: ρ = ½(I + r·σ̂) with |r| ≤ 1
✓ Mutual information: I(A:B) = S(ρ_A) + S(ρ_B) - S(ρ_AB)
✓ Quantum teleportation: Bell measurement → 2 classical bits → Pauli correction
✓ 3-qubit error code: Syndrome table with Z₁Z₂, Z₂Z₃ parity checks
