1.1.2 State and Representation#

Prompts

  • What is a quantum state as an abstract object? How does the abstract ket \(\vert \psi\rangle\) relate to its vector representation in a particular basis?

  • A qubit state can be written as \(\vert \psi\rangle = \cos(\theta/2)\vert 0\rangle + \mathrm{e}^{\mathrm{i}\phi}\sin(\theta/2)\vert 1\rangle\). Why does this require only two parameters (\(\theta\) and \(\phi\)) when we started with four real numbers?

  • Why is global phase unobservable? If we multiply a state by \(\mathrm{e}^{\mathrm{i}\gamma}\), what changes and what stays the same in measurement predictions?

  • The Bloch sphere maps each qubit state to a point on a 2-sphere. Why is this geometric picture useful? How do the angles \((\theta, \phi)\) on the sphere relate to measurement probabilities?

  • Design a measurement protocol to determine whether a qubit is in state \(\vert 0\rangle\), \(\vert +\rangle\), or \(\vert \mathrm{i}\rangle\). What bases must you measure in, and what outcomes tell you which state you have?

Lecture Notes#

Overview#

Quantum mechanics assigns to each physical system a state vector \(\vert \psi\rangle\) that encodes all information necessary to predict measurement outcomes. This section develops the mathematical language—kets, bras, inner products, and basis representations—and introduces the Bloch sphere, a geometric visualization of the space of qubit states.

Ket, Bra, and Inner Product#

Ket notation represents a quantum state as an abstract vector in a complex Hilbert space. For a two-level system (qubit), the state lives in \(\mathcal{H} = \mathbb{C}^2\).

Definition: Ket and Bra

  • The ket \(\vert \psi\rangle\) is an abstract vector in Hilbert space.

  • The bra \(\langle \psi\vert \) is the dual (Hermitian conjugate) of the ket: \(\langle \psi\vert = (\vert \psi\rangle)^\dagger\).

  • The inner product \(\langle\phi\vert \psi\rangle\) is a complex number defined by:

\[\langle\phi\vert \psi\rangle \equiv \sum_i (\phi_i)^* \psi_i\]

where \(\phi_i\) and \(\psi_i\) are components in some basis.

  • Normalization: A state is normalized if \(\langle\psi\vert \psi\rangle = 1\).

  • Orthogonality: Two states are orthogonal if \(\langle\phi\vert \psi\rangle = 0\).

The Qubit State#

A qubit is defined by the existence of two orthonormal computational basis states \(\vert 0\rangle\) and \(\vert 1\rangle\):

(2)#\[\langle 0\vert 0\rangle = 1, \quad \langle 1\vert 1\rangle = 1, \quad \langle 0\vert 1\rangle = 0\]

Any qubit state can be written as a superposition of these basis states:

(3)#\[|\psi\rangle = \alpha |0\rangle + \beta |1\rangle\]

where \(\alpha, \beta \in \mathbb{C}\) are amplitudes and the normalization condition is

(4)#\[|\alpha|^2 + |\beta|^2 = 1\]

State vs. Representation

Strictly speaking, \(\vert \psi\rangle\) is an abstract vector while \(\begin{pmatrix} \alpha \\ \beta \end{pmatrix}\) is its representation in a chosen basis. Throughout these notes we write \(\vert \psi\rangle = \begin{pmatrix} \alpha \\ \beta \end{pmatrix}\) using the equality sign, with the understanding that the right-hand side depends on the basis. If we switch to a different basis, the components change even though the state \(\vert \psi\rangle\) is the same—just as a 3D vector has different components in Cartesian vs. spherical coordinates. The same remark applies to operators and their matrix representations.

Conversion between ket and vector notation: To express \(\vert \psi\rangle = \alpha \vert 0\rangle + \beta \vert 1\rangle\) as components, write \((\alpha, \beta)\). To go the other way, interpret components \((\alpha, \beta)\) as the ket \(\alpha \vert 0\rangle + \beta \vert 1\rangle\).

Born’s Rule and Normalization#

When we measure a qubit in the computational basis, the Born rule gives the measurement probabilities:

Born Rule

If \(\vert \psi\rangle = \alpha \vert 0\rangle + \beta \vert 1\rangle\), then measurement of the qubit in the computational basis \(\{\vert 0\rangle, \vert 1\rangle\}\) yields:

\[P(\text{outcome } 0) = |\alpha|^2, \quad P(\text{outcome } 1) = |\beta|^2\]

The normalization condition \(|\alpha|^2 + |\beta|^2 = 1\) ensures probabilities sum to unity: \(P(0) + P(1) = 1\).

Global phase is unobservable: Two states \(\vert \psi\rangle\) and \(\mathrm{e}^{\mathrm{i}\gamma}\vert \psi\rangle\) (differing by a global phase \(\gamma \in \mathbb{R}\)) give identical probabilities for all measurements. For example:

\[|\psi\rangle = \alpha |0\rangle + \beta |1\rangle \quad \text{vs} \quad |\psi'\rangle = \mathrm{e}^{\mathrm{i}\gamma}(\alpha |0\rangle + \beta |1\rangle)\]

Both have \(P(0) = |\alpha|^2\) and \(P(1) = |\beta|^2\). Global phases do not affect physical predictions and can be freely removed.

Parameter Counting and Bloch Parametrization#

How many independent real parameters specify a qubit state?

Naive count: Two complex amplitudes \(\alpha, \beta\) give \(2 \times 2 = 4\) real parameters (real and imaginary parts).

Constraints:

  • Normalization: \(|\alpha|^2 + |\beta|^2 = 1\) removes 1 constraint.

  • Global phase: The freedom to multiply by \(\mathrm{e}^{\mathrm{i}\gamma}\) removes 1 parameter.

Effective count: \(4 - 1 - 1 = 2\) independent real parameters.

These two parameters naturally parametrize the qubit state:

Bloch Parametrization

Every qubit state can be written uniquely as

(5)#\[|\psi\rangle = \cos(\theta/2) |0\rangle + \mathrm{e}^{\mathrm{i}\phi} \sin(\theta/2) |1\rangle\]

where \(\theta \in [0, \pi]\) and \(\phi \in [0, 2\pi)\) are two real angles. This parametrization automatically satisfies normalization:

\[\cos^2(\theta/2) + \sin^2(\theta/2) = 1 \quad \checkmark\]

Example: Common Qubit States in Standard Form

  1. State \(\vert 0\rangle\): \(\theta = 0\), \(\phi\) arbitrary

    \[\vert \psi\rangle = \cos(0)\vert 0\rangle + \mathrm{e}^{\mathrm{i}\phi}\sin(0)\vert 1\rangle = \vert 0\rangle\]

  2. State \(\vert 1\rangle\): \(\theta = \pi\), \(\phi\) arbitrary

    \[\vert \psi\rangle = \cos(\pi/2)\vert 0\rangle + \mathrm{e}^{\mathrm{i}\phi}\sin(\pi/2)\vert 1\rangle = \mathrm{e}^{\mathrm{i}\phi}\vert 1\rangle \cong \vert 1\rangle\]

  3. State \(\vert +\rangle = \frac{1}{\sqrt{2}}(\vert 0\rangle + \vert 1\rangle)\): \(\theta = \pi/2\), \(\phi = 0\)

    \[\vert \psi\rangle = \cos(\pi/4)\vert 0\rangle + \mathrm{e}^{\mathrm{i} \cdot 0}\sin(\pi/4)\vert 1\rangle = \frac{1}{\sqrt{2}}(\vert 0\rangle + \vert 1\rangle) \quad \checkmark\]

The Bloch Sphere#

The geometric picture of qubit states is the Bloch sphere: a unit 2-sphere in 3D space where each point represents a unique qubit state.

In spherical coordinates, the Bloch parametrization becomes:

(6)#\[|\psi\rangle = \cos(\theta/2) |0\rangle + \mathrm{e}^{\mathrm{i}\phi} \sin(\theta/2) |1\rangle\]

where:

  • \(\theta \in [0, \pi]\): polar angle measured from the north pole (z-axis)

  • \(\phi \in [0, 2\pi)\): azimuthal angle around the z-axis

Orthonormal Bases

Besides the computational (Z) basis, any orthonormal pair of states defines a valid basis. We highlight the X and Y bases:

X-basis (eigenstates of \(\hat{\sigma}^x\)):

(7)#\[|+\rangle = \frac{1}{\sqrt{2}}(|0\rangle + |1\rangle), \quad |-\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle)\]

Y-basis (eigenstates of \(\hat{\sigma}^y\)):

(8)#\[|\mathrm{i}\rangle = \frac{1}{\sqrt{2}}(|0\rangle + \mathrm{i}|1\rangle), \quad |\bar{\mathrm{i}}\rangle = \frac{1}{\sqrt{2}}(|0\rangle - \mathrm{i}|1\rangle)\]

The inverse relations write the computational kets \(\vert 0\rangle\) and \(\vert 1\rangle\) in the X-basis and in the Y-basis (invert Eqs. (7) and (8)):

In the X-basis (\(\{\vert +\rangle, \vert -\rangle\}\)):

(9)#\[|0\rangle = \frac{1}{\sqrt{2}}(|+\rangle + |-\rangle), \quad |1\rangle = \frac{1}{\sqrt{2}}(|+\rangle - |-\rangle)\]

In the Y-basis (\(\{\vert \mathrm{i}\rangle, \vert \bar{\mathrm{i}}\rangle\}\)):

(10)#\[|0\rangle = \frac{1}{\sqrt{2}}(|\mathrm{i}\rangle + |\bar{\mathrm{i}}\rangle), \quad |1\rangle = -\frac{\mathrm{i}}{\sqrt{2}}(|\mathrm{i}\rangle - |\bar{\mathrm{i}}\rangle)\]

Each basis corresponds to a pair of antipodal points on the Bloch sphere.

Basis Change Example

To express a state in a different basis, substitute the inverse relations from Eq. (9) or (10). For example, using the X-basis, \(\vert \psi\rangle = \alpha \vert 0\rangle + \beta \vert 1\rangle\) becomes:

\[|\psi\rangle = \frac{\alpha + \beta}{\sqrt{2}}|+\rangle + \frac{\alpha - \beta}{\sqrt{2}}|-\rangle\]

Spin Expectation Values and the Bloch Vector#

For a qubit state \(\vert \psi\rangle\), the expectation values of the Pauli operators form a 3D unit vector:

Bloch Vector

For a state \(\vert \psi\rangle = \cos(\theta/2)\vert 0\rangle + \mathrm{e}^{\mathrm{i}\phi}\sin(\theta/2)\vert 1\rangle\), the Bloch vector is

(11)#\[\boldsymbol{n} = (\langle\hat{\sigma}^x\rangle, \langle\hat{\sigma}^y\rangle, \langle\hat{\sigma}^z\rangle) = (\sin\theta\cos\phi, \sin\theta\sin\phi, \cos\theta)\]

This is a unit vector: \(|\boldsymbol{n}| = 1\). It points from the origin directly to the state’s location on the Bloch sphere.

Physical interpretation: The Bloch vector describes the expected outcome of measuring spin in direction \(\hat{\boldsymbol{n}}\).

Note: The Pauli operators \(\hat{\sigma}^x\), \(\hat{\sigma}^y\), \(\hat{\sigma}^z\) are introduced formally in §1.1.3. For now, understand them as the three Hermitian observables whose eigenstates define the three orthonormal bases on the Bloch sphere.

Discussion

Why does the Bloch sphere fail for qubits with more than two levels?

A qubit state is specified by 2 real parameters, which naturally maps to a 2-sphere (the surface of a 3D ball). For a qutrit (3-level system), count the real parameters: \(2n\) from \(n\) complex amplitudes, minus 1 for normalization, minus 1 for global phase, gives \(2(3) - 2 = 4\) real parameters.

  • Is there a simple geometric object (like a sphere) that parametrizes all pure qutrit states?

  • How does the measurement formalism change for qubits vs. qutrits?

  • The Bloch sphere is useful because rotations in 3D (\(SU(2)\) symmetry) correspond to quantum gates. Does a similar structure exist for higher-dimensional systems?

Summary#

  • A quantum state \(\vert \psi\rangle\) is an abstract vector in a complex Hilbert space; its representation as components \((\alpha, \beta)\) depends on the chosen basis.

  • The Born rule connects amplitudes to measurement probabilities: \(P(0) = |\alpha|^2\), \(P(1) = |\beta|^2\). Global phase is unobservable.

  • A qubit state is specified by 2 real parameters: the Bloch angles \(\theta \in [0, \pi]\) and \(\phi \in [0, 2\pi)\).

  • The Bloch sphere provides a complete geometric picture of all pure qubit states as points on a unit 2-sphere.

  • Three orthonormal bases (Z, X, Y) correspond to three pairs of antipodal points on the Bloch sphere; measurement basis choice determines measurement outcome statistics.

Find the Bloch sphere angles \((\theta, \phi)\) for this state by writing it in the standard parametrization \(\vert \psi\rangle = \cos(\theta/2)\vert 0\rangle + \mathrm{e}^{\mathrm{i}\phi}\sin(\theta/2)\vert 1\rangle\).

2. Show that two states \(\vert \psi\rangle\) and \(\mathrm{e}^{\mathrm{i}\alpha}\vert \psi\rangle\) (where \(\alpha \in \mathbb{R}\)) give the same measurement probabilities for any observable. That is, show \(|\langle m \vert \psi \rangle\vert ^2 = |\langle m \vert \mathrm{e}^{\mathrm{i}\alpha}\psi \rangle\vert ^2\) for any state \(\vert m\rangle\). Why does this mean global phase is unobservable?

3. Express the state \(\vert \psi\rangle = \frac{1}{\sqrt{2}}(\vert 0\rangle + \vert 1\rangle)\) in the X-basis \(\{\vert +\rangle, \vert -\rangle\}\). Then express \(\vert \phi\rangle = \frac{1}{\sqrt{2}}(\vert 0\rangle + \mathrm{i}\vert 1\rangle)\) in the X-basis. Compute the inner product \(\langle\phi\vert \psi\rangle\) in both the Z-basis and the X-basis representations and verify you get the same result.

4. Compute the Bloch vector \(\boldsymbol{n} = (\langle \hat{\sigma}^x\rangle, \langle\hat{\sigma}^y\rangle, \langle\hat{\sigma}^z\rangle)\) for the state \(\vert \psi\rangle = \cos(\pi/8)\vert 0\rangle + \mathrm{e}^{\mathrm{i}\pi/3}\sin(\pi/8)\vert 1\rangle\). Verify that \(\vert \boldsymbol{n}| = 1\).

5. On the Bloch sphere, orthogonal quantum states correspond to antipodal points (diametrically opposite). Verify this explicitly: show that if \(\vert \psi\rangle\) has Bloch angles \((\theta, \phi)\), then the orthogonal state \(\vert \psi^\perp\rangle\) (satisfying \(\langle\psi^\perp\vert \psi\rangle = 0\)) has Bloch angles \((\pi - \theta, \phi + \pi)\).

6. A qubit state \(\vert \psi\rangle\) has Bloch vector \(\boldsymbol{n} = (\sin\theta\cos\phi, \sin\theta\sin\phi, \cos\theta)\). Show that measuring in the computational basis gives \(P(0) = \frac{1 + n_z}{2}\) and \(P(1) = \frac{1 - n_z}{2}\), where \(n_z = \cos\theta\). Interpret this geometrically: how does the measurement probability relate to the “height” of the Bloch vector?

7. The inverse relations are given in Eq. (10). Use them to express \(\vert +\rangle = \frac{1}{\sqrt{2}}(\vert 0\rangle + \vert 1\rangle)\) as a linear combination of \(\vert \mathrm{i}\rangle\) and \(\vert \bar{\mathrm{i}}\rangle\). Verify normalization and \(\langle + \vert + \rangle = 1\) using your Y-basis coefficients.

8. How many real parameters are needed to specify a general \(n\)-level quantum state (a “qudit”)? Start with the naive count of \(2n\) real numbers (from \(n\) complex amplitudes), then subtract constraints from normalization and global phase. Verify your formula gives 2 for \(n = 2\) (the qubit).
